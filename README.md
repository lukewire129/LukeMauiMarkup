# LukeMauiMarkup

### Overview

CommunityToolkit.Maui.Markup의 확장버전
 - HotReload 기본으로 적용된 버전
 - 그 외 간소화된 기능 구현

### Template install
```
dotnet new install LukeMauiMarkup.Templates::1.0.0
```
